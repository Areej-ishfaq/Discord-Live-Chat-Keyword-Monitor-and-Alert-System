# Discord Live Chat Keyword Monitor and Alert System 

A Python-based Discord monitoring bot that performs real-time keyword detection, logging, and alert visualization through a Tkinter dashboard.
This project demonstrates how automated monitoring tools can support moderation workflows and security operations while respecting permission-based access control.

## Contributors

## Features
- Real-time Discord message monitoring
- Severity-based keyword detection (Critical / Medium / Low)
- SOC-style GUI dashboard using Tkinter
- Popup alert notifications
- JSON message logging
- Audit logging system
- Alert severity distribution graph (Matplotlib)
- Permission-aware monitoring design
